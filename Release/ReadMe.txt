====================================================================================
	ИНСТРУКЦИЯ ПО УСТАНОВКЕ:
1. Перетаскиваем содержимое архива в папку "Source SDK Base 2013 Singleplayer\bin\"
2. Добавляем fgd файл из архива в конфигурацию через Tools -> Options -> Add
3. Открываем пресеты компиляции через Run Map! -> Expert...

	САМЫЕ ВАЖНЫЕ ЭТАПЫ:
4. Вам нужно создать две новые команды в левой части окна, в списке, первая должна распологаться в самом верху перед началом всей компиляции! Вторая должна быть ниже $light_exe и выше Copy File!
5. В обоих командах справа в графе Command прописываем полный путь до инструмента (например H:\SteamLibrary\steamapps\common\Source SDK Base 2013 Singleplayer\bin\PropColorCompiler.exe)
6. В графе Parameters для первой команды прописать:

$path\$file.$ext $gamedir

Для второй:

-postcompile $path\$file.$ext $gamedir

7. Инструмент полностью установлен и им можно пользоваться!
